Ferramentas

Android

O sistema operacional (OS) mais famoso dos smartphones. Uso um celular deste tipo para programar.

Acode

Um IDE mobile de código-aberto que uso para codificar.

Heliboard

Um teclado virtual de código-aberto focado em privacidade, mas que também possui recursos muitos úteis que facilitam a codificação.

Termux

Um emulador de terminal Linux livre e de código-aberto para Android.

Firefox

Um navegador que eu utilizo para conseguir depurar e ter extensões pelo celular.

Git

O sistema de controle de versão de código-aberto mais famoso de todos. Uso para gerenciar os repositórios locais e remotos.

NPM

O gerenciador de pacotes do Node.js.

Notesnook

Um serviço de anotações com criptografia de ponta-a-ponta. Útil para não esquecer dos objetivos e relembrar de algo.
